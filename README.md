## Project Overview

This project trains a Logistic Regression classifier on the Iris dataset and 
evaluates its performance. The focus is not just on the ML model — it's on 
building it with professional engineering standards.

**What this project demonstrates:**
- ✅ Clean, modular Python code with logging
- ✅ Meaningful Git commit history with conventional commits
- ✅ Feature branch workflow with Pull Requests
- ✅ Unit tests, integration tests, and mocking with pytest
- ✅ Code quality enforcement with Pylint
- ✅ Docker containerization
- ✅ Automated CI/CD with GitHub Actions
